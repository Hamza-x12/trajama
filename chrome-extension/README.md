# TARJAMA Chrome Extension

A Chromium browser extension for translating text to/from Moroccan Darija and 15+ languages.

## Features

- **Popup Translator** — Click the extension icon for a full translation interface
- **Selection Translation** — Select text on any page and click the floating "🌍 Translate" button
- **Right-Click Menu** — Right-click selected text → "Translate with TARJAMA"
- **Dark Mode** — Automatically adapts to your system theme

## Installation

1. Download/clone this `chrome-extension/` folder
2. Open `chrome://extensions` in Chrome or any Chromium browser
3. Enable **Developer mode** (toggle in the top right)
4. Click **Load unpacked** and select the `chrome-extension/` folder
5. The TARJAMA icon will appear in your toolbar

## Usage

### Popup
Click the TARJAMA icon in the toolbar → select languages → type or paste text → click **Translate**.

### Selection Translation
Select any text on a webpage → a floating **🌍 Translate** button appears above the selection → click it to see translations in Darija, English, and French.

### Context Menu
Right-click selected text → click **Translate with TARJAMA** for inline translation.

## Supported Languages

Darija, English, French, Arabic, Spanish, German, Italian, Portuguese, Turkish, Russian, Chinese, Japanese, Korean, Hindi, Amazigh

## Notes

- Requires internet connection (uses TARJAMA's translation API)
- Maximum 1000 characters per translation
- Language preferences are saved automatically
